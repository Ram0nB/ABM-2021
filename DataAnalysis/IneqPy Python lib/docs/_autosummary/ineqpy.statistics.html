ineqpy.statistics¶

Descriptive statistics.

This module contains main descriptive statistics like: mean, variance, etc.

Functions

coef_variation([variable, weights, data]) Calculate the coefficient of variation.
density([variable, weights, groups, data]) Density in percentage.
kurt([variable, weights, data]) Calculate the Kurtosis coefficient.
mean([variable, weights, data]) Calculate the mean of variable given weights.
percentile([variable, weights, data, p, …]) Calculate the value of a quantile given a variable and his weights.
skew([variable, weights, data]) Returns the asymmetry coefficient of a sample.
std_moment([variable, weights, data, param, …]) Calculate the standardized moment of order c for the variable` x` with respect to c.
var([variable, weights, data, ddof]) Calculate the variance.
